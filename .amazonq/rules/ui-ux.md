# UI/UX Guidelines

## UI Kit

- Используй только **shadcn/ui** для всех UI компонентов
- Не используй другие UI библиотеки

## Color Palette

Используй только цвета из палитры Сбера (оттенки зеленого):

- Primary: `#21A038` (основной зеленый Сбера)
- Secondary: `#1A8030` (темно-зеленый)
- Accent: `#2FC14B` (светло-зеленый)
- Success: `#28A745`
- Background: `#F5F9F6` (светлый зеленоватый)
- Dark: `#0D4D1F` (очень темный зеленый)

## Visual Style

- Стиль игры: **пиксельный (pixel art)**
- Используй пиксельные шрифты
- Применяй пиксельную графику для всех игровых элементов
- Сохраняй четкие пиксельные границы без сглаживания
