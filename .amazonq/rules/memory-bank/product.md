# Product Overview

## Project Purpose

**Cyber Security Detective** - интерактивная обучающая игра для поиска угроз кибербезопасности. Игрок выступает в роли аналитика безопасности, который анализирует различные сценарии и находит уязвимости в письмах, коде и логах.

## Value Proposition

Обучение сотрудников Сбербанка основам кибербезопасности через игровой процесс:

- Распознавание phishing атак в email
- Поиск уязвимостей в исходном коде
- Анализ логов ClickHouse для обнаружения аномалий
- Выявление подозрительной активности в сетевом трафике

## Key Features

### 1. Email Analysis Mode

Анализ подозрительных писем на наличие phishing атак. Игрок кликает на подозрительные элементы (email адреса, ссылки, вложения) и выбирает тип угрозы. Система обучает распознавать:

- Поддельные домены
- Подозрительные вложения (.exe, .zip)
- Срочные запросы данных
- Грамматические ошибки

### 2. Code Review Mode

Поиск уязвимостей в исходном коде. Игрок выделяет подозрительные строки и определяет тип уязвимости:

- SQL Injection
- XSS (Cross-Site Scripting)
- Hardcoded credentials
- Missing validation
- Insecure authentication

### 3. ClickHouse Logs Mode

Анализ больших объемов логов для поиска аномалий. Игрок пишет SQL запросы для обнаружения:

- Brute force атак
- Data exfiltration
- SQL injection попыток
- DDoS атак
- Privilege escalation

### 4. Progression System

- 4 уровня сложности (Junior Analyst → CISO Challenge)
- Система очков и комбо
- Достижения и рейтинги
- Обучающие материалы после каждой угрозы

### 5. Visual Design

Виртуальное рабочее место аналитика безопасности:

- Монитор с 3D эффектом (CSS transforms)
- Современная клавиатура (CSS/SVG дизайн)
- Цветовая палитра Сбера (зеленые оттенки)
- Анимации успеха/ошибки (Framer Motion)

## Target Users

- **Сотрудники Сбербанка** (основная аудитория)
- **IT специалисты** (начинающие)
- **Студенты** (информационная безопасность)
- **Все, кто хочет** изучить кибербезопасность

## Use Cases

### Корпоративное обучение

Компании используют игру для обучения сотрудников основам кибербезопасности в интерактивном формате.

### Самообразование

IT специалисты практикуют навыки обнаружения угроз в безопасной игровой среде.

### Учебные заведения

Преподаватели используют игру как практический инструмент для курсов по информационной безопасности.

## Success Metrics

### Игровые метрики

- Процент прохождения уровней
- Средняя точность обнаружения угроз
- Среднее время прохождения
- Количество найденных угроз

### Обучающие метрики

- Улучшение навыков со временем
- Типы угроз, которые чаще пропускают
- Эффективность обучающих материалов
