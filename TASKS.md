# Agent-game

## 🎮 Cyber Security Detective

Интерактивная обучающая игра для поиска угроз кибербезопасности.

---

## 🛠️ Необходимые инструменты и технологии

### Основной стек

- **React 18** + **TypeScript** - основа приложения
- **Vite** - сборщик и dev-сервер
- **shadcn/ui** - UI компоненты (обязательно по правилам)
- **Tailwind CSS** - стилизация
- **Framer Motion** - анимации и эффекты
- **Lucide React** - иконки

### Дополнительные библиотеки

- **Recharts** - графики для ClickHouse режима
- **Monaco Editor** - SQL редактор с подсветкой синтаксиса
- **Zustand** - глобальный стор (через провайдер)
- **React Router** - навигация между экранами
- **React Hook Form** - управление формами (если нужно)

### Dev инструменты

- **ESLint** - линтинг JavaScript/TypeScript
- **Stylelint** - линтинг CSS
- **Prettier** - форматирование кода
- **Husky** + **lint-staged** - pre-commit хуки
- **TypeScript** - проверка типов

### Деплой

- **GitHub Actions** - CI/CD
- **GitHub Pages** - хостинг

## 🎯 Приоритеты для MVP (v1.0)
