# 🎮 Cyber Security Detective

## Концепция игры

**Cyber Security Detective** - интерактивная обучающая игра для поиска угроз кибербезопасности. Игрок выступает в роли аналитика безопасности, который анализирует различные сценарии и находит уязвимости.

---

## 🎯 Цель игры

Обучить сотрудников Сбербанка основам кибербезопасности через игровой процесс:

- Распознавание phishing атак
- Поиск уязвимостей в коде
- Анализ логов и обнаружение аномалий
- Выявление подозрительной активности

## 🎯 Режимы игры

### 1️⃣ Email Analysis Mode

**Описание**: Анализ подозрительных писем на наличие phishing атак.

**Интерфейс:**

```
┌────────────────────────────────────────┐
│ От: support@paypa1.com ⚠️              │
│ Кому: employee@company.com             │
│ Тема: Urgent: Verify your account     │
├────────────────────────────────────────┤
│ Dear user,                             │
│                                        │
│ Your account will be suspended.        │
│ Click here: http://bit.ly/sus123      │ ← Кликабельно
│                                        │
│ Best regards,                          │
│ PayPal Security Team                   │
└────────────────────────────────────────┘

[Отметить как угрозу] [Пропустить]
```

**Механика:**

- Игрок кликает на подозрительные элементы (email, ссылки, вложения)
- При клике появляется диалог: "Почему это угроза?"
  - ☑️ Подозрительный домен (paypa1 вместо paypal)
  - ☑️ Срочность и давление
  - ☑️ Сокращенная ссылка
- Правильный выбор → **+100 очков**
- Неправильный выбор → **-50 очков**

**Типы угроз:**

- Поддельные домены
- Подозрительные вложения (.exe, .zip)
- Срочные запросы данных
- Грамматические ошибки
- Несоответствие отправителя

---

### 2️⃣ Code Review Mode

**Описание**: Поиск уязвимостей в исходном коде.

**Интерфейс:**

```javascript
// API Endpoint для логина
app.post('/login', (req, res) => {
  const { username, password } = req.body;

  // SQL запрос
  const query = `SELECT * FROM users
                 WHERE username = '${username}'
                 AND password = '${password}'`;  ← УГРОЗА!

  db.query(query, (err, result) => {
    if (result.length > 0) {
      res.json({ token: generateToken() });
    }
  });
});
```

**Механика:**

- Игрок наводит курсор на строки кода
- Подозрительные строки можно выделить
- При выделении выбрать тип угрозы:
  - 🔴 SQL Injection
  - 🟡 XSS (Cross-Site Scripting)
  - 🟠 Hardcoded credentials
  - 🔵 Missing validation
  - 🟣 Insecure authentication
- Показывается объяснение + правильное решение

**Типы уязвимостей:**

- SQL Injection
- XSS (Cross-Site Scripting)
- CSRF (Cross-Site Request Forgery)
- Hardcoded secrets
- Слабое шифрование
- Отсутствие валидации
- Path traversal
- Command injection

---

### 3️⃣ ClickHouse Logs Mode (самый сложный)

**Описание**: Анализ больших объемов логов для поиска аномалий и атак.

**Интерфейс:**

#### Верхняя часть - SQL редактор:

```sql
SELECT
  ip_address,
  COUNT(*) as attempts,
  status_code
FROM access_logs
WHERE timestamp > now() - INTERVAL 1 HOUR
GROUP BY ip_address
HAVING attempts > 100
ORDER BY attempts DESC
```

[▶️ Выполнить запрос]

#### Нижняя часть - Результаты:

**Таблица (shadcn Table):**

```
┌─────────────────┬──────────┬─────────────┬──────────┐
│ IP Address      │ Attempts │ Status Code │ Action   │
├─────────────────┼──────────┼─────────────┼──────────┤
│ 192.168.1.100   │ 1,247 🔴│ 401         │ [Flag]   │
│ 10.0.0.50       │ 856   🟡│ 200         │ [Flag]   │
│ 172.16.0.1      │ 45    🟢│ 200         │ [Flag]   │
└─────────────────┴──────────┴─────────────┴──────────┘
```

**График (Recharts):**

```
Requests per minute
  │
1k│    ╱╲
  │   ╱  ╲___
500│__╱       ╲___
  │
  └─────────────────
   00:00  01:00  02:00  03:00 ← Spike в 01:00!
```

**Механика:**

- Игрок пишет SQL запросы для поиска аномалий
- Есть предустановленные запросы-шаблоны (подсказки)
- Нужно найти:
  - 🔴 **Brute force** - много неудачных попыток входа (401) с одного IP
  - 🟡 **Data exfiltration** - большие объемы данных с одного IP
  - 🟠 **SQL injection** - подозрительные символы в параметрах запросов
  - 🔵 **DDoS атака** - необычный spike в трафике
  - 🟣 **Privilege escalation** - доступ к запрещенным ресурсам
- При нахождении угрозы → кнопка **[Flag as Threat]**

**Типы логов:**

- Web server access logs (nginx/apache)
- Authentication logs (failed/successful logins)
- API request logs
- Database query logs
- Network traffic logs

**Примеры сценариев:**

1. "В базе 10M записей за последний час. Найдите IP, совершающий brute force атаку"
2. "Обнаружьте SQL injection попытки в параметрах запросов"
3. "Найдите аномальный spike в трафике в 3:00 AM"
4. "Выявите IP, который скачал более 10GB данных за час"

## 🎯 Система прогрессии

### Уровни сложности:

#### **Level 1: Junior Analyst** (Легко)

- 3-5 очевидных угроз
- Подсказки доступны (неограниченно)
- Таймер: 10 минут
- Режимы: Email + Code Review

#### **Level 2: Security Specialist** (Средне)

- 5-7 угроз, некоторые скрытые
- 2 подсказки
- Таймер: 7 минут
- Режимы: Email + Code Review + ClickHouse (простые запросы)

#### **Level 3: Senior Expert** (Сложно)

- 7-10 угроз, хорошо замаскированные
- 1 подсказка
- Таймер: 5 минут
- Режимы: Все режимы
- ClickHouse: сложные запросы с JOIN

#### **Level 4: CISO Challenge** (Эксперт)

- 10+ угроз, комбинированные атаки
- Нет подсказок
- Таймер: 3 минуты
- Все режимы одновременно
- Цепочки атак (phishing → code exploit → data exfiltration)

## 🏆 Система очков и наград

### Очки:

- **Найдена угроза**: +100 очков
- **Ложное срабатывание**: -50 очков
- **Пропущена угроза**: -100 очков
- **Бонус за скорость**: +50 за каждую минуту до таймера
- **Комбо**: +25 за каждую последовательную находку без ошибок

### Достижения:

- 🥇 **"Орлиный глаз"** - найти все угрозы без ошибок
- ⚡ **"Скоростной анализ"** - завершить уровень за 3 минуты
- 🎯 **"SQL Мастер"** - найти 10 угроз в ClickHouse режиме
- 🛡️ **"Защитник"** - пройти 10 уровней
- 🔥 **"Комбо x10"** - найти 10 угроз подряд без ошибок
- 🌟 **"Перфекционист"** - пройти уровень с 100% точностью
- 💎 **"Легенда"** - пройти все уровни на эксперте

### Рейтинговая система:

- **Trainee** (0-500 очков)
- **Analyst** (500-1500 очков)
- **Specialist** (1500-3000 очков)
- **Expert** (3000-5000 очков)
- **Master** (5000-10000 очков)
- **Legend** (10000+ очков)

## 🎨 Визуальные эффекты

### При нахождении угрозы:

- ✅ Зеленая анимация (Framer Motion)
- Звук "успех" (опционально)
- +100 очков летят вверх
- Монитор слегка мигает зеленым (#21A038)
- Прогресс-бар обновляется

### При ошибке:

- ❌ Красная анимация
- Звук "ошибка" (опционально)
- -50 очков
- Монитор мигает красным
- Встряска экрана (shake effect)

### При завершении уровня:

- 🎉 Конфетти (зеленые оттенки)
- Модальное окно со статистикой:
  ```
  ┌─────────────────────────────┐
  │   🎉 УРОВЕНЬ ЗАВЕРШЕН!      │
  ├─────────────────────────────┤
  │ Найдено угроз: 5/5          │
  │ Точность: 85%               │
  │ Время: 04:23                │
  │ Бонус за скорость: +200     │
  │                             │
  │ ⭐ Итого: 1,450 очков       │
  ├─────────────────────────────┤
  │ [Следующий уровень]         │
  └─────────────────────────────┘
  ```

### Анимации монитора:
